Unang Program: Hello World

Ito ang pinakasimpleng program sa C - ang traditional first program ng lahat ng programmers!

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }
Paliwanag line-by-line:
‚Ä¢ #include <stdio.h> - naglo-load ng standard input/output library para magamit ang printf()
‚Ä¢ int main() - main function, dito nagsisimula ang program execution
‚Ä¢ printf() - function para mag-print/display sa screen
‚Ä¢ \n - newline character, para lumipat sa bagong line
‚Ä¢ return 0 - sinasabi na successful ang program (0 = no errors)

Step-by-Step: Gumawa ng Hello World

1 Buksan ang Text Editor

Open ang Code::Blocks o any text editor mo. Create new file (File ‚Üí New ‚Üí Empty File).

2 Type ang Header

Magsimula sa #include <stdio.h> - ito ay library para sa input/output functions. Palaging nasa top ng file ito.

3 Create Main Function

Type: int main() { } - lahat ng code ay nasa loob ng curly braces. Ito ang entry point ng program.

4 Add Printf Statement

Sa loob ng main, type: printf("Hello, World!\n"); - Huwag kalimutan ang semicolon (;) sa dulo!

5 Return Statement

Lagay ang return 0; para sabihin na successful ang program. Ito ay standard practice.

6 Save at Compile

Save as hello.c (importante ang .c extension) at compile using F9 or Build and Run button.

Printf at Scanf - Input/Output

Ang printf() ay para mag-display, ang scanf() ay para mag-receive ng user input:

#include <stdio.h> int main() { int edad; char pangalan[50]; float height; // Get user input printf("Ano ang iyong pangalan? "); scanf("%s", pangalan); // No & for strings printf("Ilang taon ka na? "); scanf("%d", &edad); // Use & for int, float, etc. printf("Gaano ka katangkad (feet)? "); scanf("%f", &height); // Display with formatting printf("\n=== YOUR INFO ===\n"); printf("Kumusta %s!\n", pangalan); printf("Edad: %d taon\n", edad); printf("Height: %.2f feet\n", height); return 0; }
‚ö†Ô∏è Important Scanf Rules:
‚Ä¢ Gumamit ng & (ampersand) before variable name para sa int, float, char
‚Ä¢ HUWAG gumamit ng & para sa strings/arrays
‚Ä¢ Gumamit ng tamang format specifier na match sa data type
‚Ä¢ Mag-ingat sa spaces - scanf() stops at whitespace for %s

Recursion - Function Calling Itself

Ang recursion ay technique kung saan tumatawag ang function sa sarili niya:

#include <stdio.h> // Recursive factorial int factorial(int n) { if (n == 0 || n == 1) { return 1; // Base case } return n * factorial(n - 1); // Recursive call } // Iterative factorial (for comparison) int factorialIterative(int n) { int result = 1; for (int i = 2; i <= n; i++) { result *= i; } return result; } // Fibonacci sequence int fibonacci(int n) { if (n <= 1) { return n; } return fibonacci(n-1) + fibonacci(n-2); } int main() { printf("Factorial of 5: %d\n", factorial(5)); printf("Fibonacci series (0-9): "); for (int i = 0; i < 10; i++) { printf("%d ", fibonacci(i)); } printf("\n"); return 0; }
‚ö†Ô∏è Recursion Tips:
‚Ä¢ Always have a BASE CASE para hindi infinite recursion
‚Ä¢ Recursion uses more memory - stack overflow possible
‚Ä¢ Hindi lahat ng problems ay mas maganda sa recursion
‚Ä¢ Iterative solution is usually faster

Pass by Value vs Pass by Reference

// Pass by value - copy of value void changeValue(int x) { x = 100; // Changes local copy only } // Pass by reference - using pointers void changeValueRef(int *x) { *x = 100; // Changes original variable } int main() { int num = 50; changeValue(num); printf("After pass by value: %d\n", num); // Still 50 changeValueRef(&num); printf("After pass by reference: %d\n", num); // Now 100 return 0; }

üìä Arrays

Ano ang Array?

Ang array ay collection ng same data type stored sa contiguous memory. Parang filing cabinet na may numbered drawers (indexes).

// Declare array ng 5 integers int grades[5]; // Initialize array with values int numbers[5] = {10, 20, 30, 40, 50}; // Partial initialization int arr[5] = {1, 2}; // Rest are 0 // Automatic size int values[] = {5, 10, 15, 20}; // Size is 4 // Access elements (index starts at 0) printf("First: %d\n", numbers[0]); // 10 printf("Third: %d\n", numbers[2]); // 30 printf("Last: %d\n", numbers[4]); // 50 // Modify element numbers[1] = 25; printf("Second: %d\n", numbers[1]); // 25
Array Facts:
‚Ä¢ Index starts at 0 (zero-based indexing)
‚Ä¢ Size must be constant (cannot change after declaration)
‚Ä¢ Array name is actually a pointer to first element
‚Ä¢ Adjacent memory locations - fast access
‚Ä¢ Bounds are not checked - be careful with index!

Step-by-Step: Student Grade System

#include <stdio.h> int main() { // Step 1: Declare array and variables int grades[5]; int sum = 0; float average; int highest, lowest; // Step 2: Input grades printf("=== STUDENT GRADE SYSTEM ===\n"); printf("Enter 5 student grades:\n"); for (int i = 0; i < 5; i++) { printf("Student %d: ", i+1); scanf("%d", &grades[i]); // Validate input while (grades[i] < 0 || grades[i] > 100) { printf("Invalid! Enter again (0-100): "); scanf("%d", &grades[i]); } } // Step 3: Calculate sum for (int i = 0; i < 5; i++) { sum += grades[i]; } // Step 4: Calculate average average = sum / 5.0; // Step 5: Find highest and lowest highest = lowest = grades[0]; for (int i = 1; i < 5; i++) { if (grades[i] > highest) { highest = grades[i]; } if (grades[i] < lowest) { lowest = grades[i]; } } // Step 6: Display results printf("\n=== RESULTS ===\n"); for (int i = 0; i < 5; i++) { printf("Student %d: %d\n", i+1, grades[i]); } printf("Sum: %d\n", sum); printf("Average: %.2f\n", average); printf("Highest: %d\n", highest); printf("Lowest: %d\n", lowest); // Step 7: Count passed/failed int passed = 0, failed = 0; for (int i = 0; i < 5; i++) { if (grades[i] >= 75) { passed++; } else { failed++; } } printf("Passed: %d\n", passed); printf("Failed: %d\n", failed); return 0; }

Step-by-Step: String Operations

#include <stdio.h> #include <string.h> #include <ctype.h> int main() { // Step 1: Declare strings char firstName[50], lastName[50], fullName[100]; char email[100]; // Step 2: Get input printf("Enter first name: "); scanf("%s", firstName); printf("Enter last name: "); scanf("%s", lastName); // Step 3: Create full name strcpy(fullName, firstName); strcat(fullName, " "); strcat(fullName, lastName); // Step 4: Convert to uppercase char upperName[100]; strcpy(upperName, fullName); for (int i = 0; upperName[i]; i++) { upperName[i] = toupper(upperName[i]); } // Step 5: Create email strcpy(email, firstName); strcat(email, "."); strcat(email, lastName); strcat(email, "@email.com"); // Convert email to lowercase for (int i = 0; email[i]; i++) { email[i] = tolower(email[i]); } // Step 6: Display results printf("\n=== RESULTS ===\n"); printf("Full name: %s\n", fullName); printf("Uppercase: %s\n", upperName); printf("Email: %s\n", email); printf("Name length: %d characters\n", strlen(fullName)); return 0; }

Sorting Arrays - Bubble Sort

#include <stdio.h> int main() { int arr[5] = {64, 34, 25, 12, 22}; int n = 5; printf("Before sorting: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } // Bubble sort algorithm for (int i = 0; i < n-1; i++) { for (int j = 0; j < n-i-1; j++) { if (arr[j] > arr[j+1]) { // Swap elements int temp = arr[j]; arr[j] = arr[j+1]; arr[j+1] = temp; } } } printf("\nAfter sorting: "); for (int i = 0; i < n; i++) { printf("%d ", arr[i]); } return 0; }

üëâ Pointers

Ano ang Pointer?

Ang pointer ay special variable na nag-store ng memory address ng ibang variable. Ito ay powerful at dangerous feature ng C!

int number = 42; int *ptr; // pointer declaration ptr = &number; // store address ng number printf("Value: %d\n", number); // 42 printf("Address: %p\n", &number); // memory address printf("Ptr stores: %p\n", ptr); // same address printf("Value via ptr: %d\n", *ptr); // 42 (dereferencing)
Important Symbols:
‚Ä¢ * sa declaration (int *ptr) = pointer type
‚Ä¢ & = address-of operator (get address)
‚Ä¢ * sa code (*ptr) = dereference (get value at address)

Practical Example: Swap Function

void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } int main() { int x = 5, y = 10; printf("Before swap: x=%d, y=%d\n", x, y); swap(&x, &y); printf("After swap: x=%d, y=%d\n", x, y); return 0; }
